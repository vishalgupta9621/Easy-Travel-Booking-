## üîç **Search & Browse Services**

### **Hotel Search**
1. **Select "Hotels"** from the travel options
2. **Enter search criteria**:
   - Destination city
   - Check-in date
   - Check-out date
   - Number of guests
3. **Click "Search Hotels"**
4. **Browse results** with filters:
   - Price range
   - Star rating
   - Amenities
   - Location

### **Flight Search**
1. **Select "Flights"** from travel options
2. **Enter flight details**:
   - Origin city
   - Destination city
   - Departure date
   - Return date (if round trip)
   - Number of passengers
   - Class preference
3. **Click "Search Flights"**
4. **Filter results** by:
   - Price
   - Airline
   - Departure time
   - Duration

### **Train Search**
1. **Select "Trains"** from travel options
2. **Enter journey details**:
   - From station
   - To station
   - Journey date
   - Class preference
3. **Click "Search Trains"**
4. **Filter by**:
   - Departure time
   - Train type
   - Class availability
   - Duration

### **Bus Search**
1. **Select "Buses"** from travel options
2. **Enter travel details**:
   - From city
   - To city
   - Travel date
   - Bus type preference
3. **Click "Search Buses"**
4. **Filter results** by:
   - Departure time
   - Bus operator
   - Bus type (AC/Non-AC)
   - Price range

### **Package Search**
1. **Select "Packages"** from travel options
2. **Enter package criteria**:
   - Destination
   - Travel dates
   - Number of travelers
   - Budget range
   - Package type
3. **Browse available packages**
4. **Filter by**:
   - Duration
   - Price
   - Destinations included
   - Package type

## ‚ùì **Troubleshooting**

### **Common Issues**

#### **Login Problems**
- **Check username/email** spelling
- **Verify password** is correct
- **Clear browser cache** and cookies
- **Try different browser**
- **Reset password** if needed

#### **Search Not Working**
- **Check internet connection**

- **Clear browser cache**
- **Refresh the page**

#### **Payment Issues**
- **Check payment details** are correct
- **Ensure sufficient balance**
- **Try different payment method**
- **Check bank/card restrictions**
- **Contact payment provider**

#### **Booking Confirmation Missing**
- **Check email spam folder**
- **Verify email address** is correct
- **Check booking history** in account
- **Contact support** with payment details

#### **Ticket Download Problems**
- **Allow popups** in browser settings
- **Try different browser**
- **Check PDF viewer** is installed
- **Use email option** as alternative
